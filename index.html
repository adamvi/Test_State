About Vistat

Statistical graphics are powerful – your eyes will jump to the graph below immediately and skip this paragraph automatically. When we see a nice graph, we often wonder how it was made (e.g. this one via xkcd).

Vistat aims to provide working recipes for statistical graphics by building them from source. The website is based on Github/Jekyll, and graphs are generated dynamically through the R package knitr, hence reproducibility is guaranteed, and readers can see the source code at the same time.

How to contribute

You can fork the repository on Github, add your article to the _source directory and submit a pull request to us. We will run your code and upload images. For more details, see about.
